Modals

Guiding Principles

Info, Warning and Critical modals present themselves whenever the users attention needs to be taken off the screen they are currently on. They produce a faded background behind the popup. Alerts are presented when a taken action requires little to no notice, such as a successful upload, or info about elements on a page. If a users screen is focused in the center of a large data table i.e. more than 10 rows, a popup will always be used, so that when the user closes the window they will not lose their place in the page.
All alerts can only be dismissed by clicking the close button in the upper right corner of the colored box. List groups are used when the user needs to make one or more decicions inside a modal. Modals may vary in width and height erratically depending on window content. All sizes that do not fall within "Info, Warning, and Critical are determined by 30px of container padding. All modal close buttons use the HTML Glyph "& # 9587"
